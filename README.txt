МАРИО 🍄 — Платформ Тоглоом
============================

ХЭРХЭН АЖИЛЛАДАГ:
- A/D буюу ⬅️➡️ сум: зүүн/баруун хөдөлгөө
- SPACE: үсрэлт
- P: пауз/үргэлжүүлэх

ЗОРИЛГО:
✅ Эхэлээ (🚩) хүрэх
⭐ Мөнгө цээ цуглуулах (нэмэх оноо)
🔴 Өтрүүдийг сойлох (амьд алдах)
❤️ Амьдыг хорвуулах (3 ба амьд)

ЯЛАЛТ:
- Флаг хүрвэл дараагийн түвшинд орно
- Эх түвшин дахь өтрүүд нэмэгддэг
- Цорын ганц аливаа түвшин хүрэх хөтөлбөрөөс үлгэрэх боломжгүй

ТӨВШНҮҮД:
- Түвшин 1: 2 өтөр
- Түвшин 2: 3 өтөр (хурдан)
- Түвшин 3+: Илүү өтөр, илүү хурдан

ОРУУЛАХ:
- Веб хөтчөө нээж mario.html сонго
- Эсвэл мөрийн үйлдлээр нээ: python -m http.server 8000 дараа http://localhost:8000/mario.html
